# Simple Game App

## Project Overview

The **Simple Game App** is a mobile application developed using **React Native with Expo**. The application includes two interactive games: a **Tic-Tac-Toe game** and a **Multiple-Choice Quiz**. The app also integrates **Supabase authentication and database services** to support user login, logout, and session handling.

This project was developed as part of a team assignment to demonstrate mobile application planning, architecture design, UI prototyping, and implementation skills.

## Authentication System

The application uses **Supabase Authentication** to allow users to:

- Create an account
- Log in securely
- Log out safely
- Maintain active sessions while using the application

Authentication ensures that users can access protected features inside the app.

## Database Integration

Supabase database is connected to the application to support:
